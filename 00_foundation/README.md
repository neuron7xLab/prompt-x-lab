# 00 — Foundation

The primitives every higher-layer module inherits.

| Module | Purpose |
| --- | --- |
| [`identity-primitive.md`](identity-primitive.md) | Declare *who* the model is and what it will refuse to be. |
| [`constraint-primitive.md`](constraint-primitive.md) | Declare *what must not happen*. |
| [`output-primitive.md`](output-primitive.md) | Declare *the exact shape* of the answer. |

These three blocks compose the minimal viable prompt. Everything else in this repository is a specialization of them.
